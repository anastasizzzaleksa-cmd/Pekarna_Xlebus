# ИС «Молочный комбинат»

Информационная система для автоматизации учёта заказов покупателей, расчёта себестоимости продукции и управления пользователями с ролевой моделью доступа.


## Описание проекта

Проект разработан для ООО Молочный комбинат «Полесье» и предназначен для:

- Ведения заказов покупателей
- Расчёта полной стоимости заказа с учётом норм расхода материалов
- Разграничения доступа пользователей (Администратор / Пользователь)
- Защиты от несанкционированного доступа через капчу-пазл
- Валидации данных клиентов через внешний API-эмулятор

---

## Функциональные возможности

### Для всех пользователей
- Авторизация по логину/паролю
- Сборка капчи-пазла для входа
- Валидация ФИО через API эмулятор
- Сохранение результатов валидации в ТестКейс.docx

### Для Администратора
- Добавление новых пользователей (с проверкой уникальности логина)
- Редактирование данных пользователей
- Снятие блокировки с учётной записи
- Просмотр списка всех пользователей
- Изменение ролей пользователей

### Системные функции
- Автоматическая блокировка после 3 неудачных попыток входа
- Расчёт стоимости заказа на основе спецификаций
- Интеграция с эмулятором внешнего API

## Технологии

 Компоненты и Технология -  
Язык программирования: C# (.NET Framework 4.7.2 / .NET 6) 
Интерфейс Windows Forms / WPF 
База данных: PostgreSQL 14+ / MySQL 8+ 
Доступ к БД : ADO.NET (Npgsql / MySql.Data) 
API-запросы: HttpClient 
Работа с Word : Microsoft.Office.Interop.Word 
Шифрование паролей : bcrypt / crypt() 
Капча : Пользовательский компонент (пазл)
